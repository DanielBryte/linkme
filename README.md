## How to get started, test locally

First, clone the site

run git clone <repository_url> in terminal

```bash
git clone https://github.com/DanielBryte/linkme.git
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
